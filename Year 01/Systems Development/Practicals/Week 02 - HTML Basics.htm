Introduction

This first worksheet is designed to get you to write simple HTML 4 documents on the Debian Linix system using the gedit (or whatever editor you like) and then to view them using the iceweasel (AKA Firefox) browser. We will discuss these in order.

Your tutor will help you by demonstrating what is required and also answer any questions you may have.

What you must do

If you think you have gone wrong at any point - ask your tutor for help

Moving around the Unix system

Use a console to enter Unix commands – open the terminal as in the workshop last week.

Change to your home directory (folder) then to your HTML directory which called public_html

cd

cd public_html

Check that you are in the correct directory using:

pwd

Using gedit editor

You can use this text editor to write text files such HTML files – you should have tried this out last week.

Make sure you are in your public_html directory and then type in:

gedit first.html

Type into the gedit window your first html program as shown below (actually it is better to Copy and Paste - see end of this worksheet – if you do this you’ll need to start Iceweasel and load the workshop in Unix)

<html>

<head>

<title>My First HTML</title>

</head>

<body bgcolor=white>

<h1>This is My First HTML</h1>

Hello my name is PutYourNameHere

<hr>

Bye

</body>

</html>

Then save this file.

You should always save your files frequently when editing. (why?)

View your file with iceweasel

Start iceweasel as you did last week.

When iceweasel fires up use File -> Open to read in your file public_html/first.html. You should see the iceweasel version of this:

Hooray!

Cutting and Pasting on Unix (note - may not apply to all UWE computers)

(This is a lot easier than on PC Windows)

• wipe the required text with the Left Mouse Button (LMB) held down

• position the cursor using the mouse

• drop the text by pressing the Middle Mouse Button (MMB)

To cut and paste- wipe, position, drop
