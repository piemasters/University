CGI and Cookies

Start by modifying the test.cgi program you downloaded last week
Firstly, get the program to generate the initial form (i.e. the form called htmltest.htm last week - see the lecture for details of how to modify the program) so that the user initially loads the test.cgi program rather than the htmltest.htm form
Check out the one-page reference at: http://www.deakin.edu.au/~agoodman/scc381/perlcgi.html  (if asked, hit 'Proceed' to view the document)
Now try coding a simple calculator - get the user to enter two numbers and select (perhaps via a radio button group) one of four operations (+,-,*,/).  When the form is submitted your CGI program should carry out the calculation and return the result, along with a new blank form ready for the next calculation.  If you are clever you will only need one set of code to generate both the initial and response forms - see the lecture for a CGI template
Now try adding a persistent memory using cookies (add buttons or checkboxes to save a result and load the saved value)
Experiment, experiment...!!!
