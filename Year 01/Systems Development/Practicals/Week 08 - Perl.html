Perl

The aim of this worksheet is:

Give you the opportunity to learn to program in Perl
Learn to write and run Perl programs.
Encourage you to self-learn by following references to further help
Develop the skills required for your assignment

Review

You should make sure that you are familiar with the Perl lecture notes.

References

Make sure you follow the references to further help in the lecture notes and bookmark them - they you may want to spend some time on them over Xmas.

Perl programs

SAVE: You will need to save the 10 Perl programs demonstrated in the lecture (use the right mouse button on the links)

PREPARE: You should need to set the correct file permission using chmod (755 is OK)

RUN: If you have set up the programs properly you should just type the programs name to run it e.g. buy.pl - otherwise type perl buy.pl

Exercise

For each program in turn:

Save the program, set the file permissions and run it
Put the program into an editor and read it
Using your book, the lecture notes and web references work through the program making sure you understand each line
Experiment with the program - try to make it do more!

DO NOT go onto the next program until you are sure you understand the current one.

Here are links to the programs:
scalar1.pl input.pl match.pl string.pl loop1.pl
file.pl buy.pl array.pl push.pl hash.pl

You will also need the shop.txt input file until you write your own.

To learn any programming language you have to write lots of programs - try and start small and add features to your programs.  If you get stuck find out what to do from a book, the internet, your PAL leader or the module staff

Enjoy!
