UFCE47-20-1 - Systems Development

Workshop 1

Author: Julian Webb

This week, you should familiarise yourself with the use of the terminals in your assigned laboratory.  Your tutor will help you get started - please ask if you get stuck or don't understand what to do.

At the end of this session you should be able to:

  1. Log on to the UWE Unix system
  2. Make use of :

    3. i) Iceweasel (a rebranded version of Iceweasel) to browse the Internet

    ii) Use editors to create simple text files (you will meet a more sophisticated editor next week)

  3. Log off the Unix system

Notes

  1. To log in:
  2. Start Windows
  3. Go to Start->All Programs->Xming->Xming (NB for some computers Start->All Programs->Network Clients->Xming->Xming)
    This starts the Xming client that allows you to connect to Unix
    Select a 'GNOME' session from the 'session type' dropdown in the extreme lower-left corner of the Xming screen by clicking on the O to the right of 'GNOME'
  4. Enter your CEMS username and password
    both user name and password are case-sensitive (i.e. it matters whether you type upper or lower case letters) - if you can't log in, check you have not got the 'Caps Lock' key on
    when typing, make sure you distinguish between '1' (the digit) and 'l' (lower-case L) and between '0' (the digit) and 'O' (uppercase O)
  5. Though you can now see a Windows-like environment you need to get used to using 'raw' Unix commands, so...
  6. ...to start Iceweasel:

    7. i) activate a terminal window (via applications->accessories->terminal).  Try resizing, and maximising/minimising the window and generally spend some time getting familiar with the effects of right and left clicking on various areas of the window.

    ii) type 'Iceweasel &' in the terminal window. (The '&' after Iceweasel means that the Iceweasel window will open and allow you to enter further commands into the Console window (try it without the '&' to see what happens).  Iceweasel will start up.

  7. check out the UFCE47-20-1 homepage and web site by typing in the URL (Uniform Resource Locator): http://www.cems.uwe.ac.uk/~jtwebb/ufce47-20-1

    8. (the '~' character is called a 'tilde' and here is a shortcut to Julian's public_html directory)

    Bookmark it for future ease of access.

  8. Make sure you know how to access your email via the UWE student email system
  9. start the 'gedit' editor in a similar way to Iceweasel (or from the applications' menu). 
  10. gedit is good for advanced editing.  However, it is a very good idea to also know a basic, commonly available Unix editor like 'nano' (AKA 'pico' on other Unix systems) as not all systems have anything more sophisticated.
  11. Try Googling 'Unix Tutorial' and try out some basic Unix commands (a good starting point would be ls (which lists files etc. in the current directory; cd - which changes directory; and mkdir and rmdir - these make and delete directories. A nice idea is to open the File Manager while you try these out - you can then see the effects of what you are doing visually).  Everything you learn about Unix commands may be very useful to you later in your careers (because not that many students know anything about them)!
  12. to log off Unix, go to 'System' and select 'log out'
