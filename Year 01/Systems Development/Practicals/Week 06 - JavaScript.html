Exercise 1

  • Make an html page in your own web space by copying the first simple example from the lecture note (Mr Pudding says ..). I have saved this HTML file as hiThere.html
  • Save the page and give it correct permissions
  • Check that you can load the page into the browser using http: protocol
  • Edit the page to make Mr Pudding tell you how he is feeling today!

Exercise 2

  • Make a page with the next example (Ms Pud speaks ..). Saved as: hiThereHead.html
  • Check that you can load the page into the browser using http: protocol
  • Write another function called sayHello - when you call sayHello with a name (e.g. Jay) it should write out "Hello Jay"

Exercise 3

  • Do the same with the popups example - you will need to save or link to image files. You can read the HTML source to find where this image file is! Saved as: popups.html
  • Check that you can load the page into the browser using http: protocol
  • When you hover the mouse over the image some text appears - where does this come from? Change this text to read "Click me nicely!"
  • When you click Cancel after being asked to talk the pop box says Sob ..., instead of this pop up another box saying "I insist you speak to me!"

Exercise 4

  • Run the rollover example in your own web space- Saved as: rollover.html
  • Change the images for 2 that you have made or got from elsewhere (they are MANY free images on the internet - use a search engine)

Exercise 5 (DIFFICULT)

  • Finally - back to the first example - days to xmas... Saved as: xmas.html
  • Change this so that it calculates how many days to your birthday!
  • What happens if your birthday has already passed this year? After you have set your birthday

    xmas.setFullYear(today.getFullYear());

    you can add the following code to add one to the year if your birthday has already passed!

    if ( xmas.getTime() < today.getTime() ) {
      //xmas has gone this year
      xmas.setFullYear(today.getFullYear() +1 );
    }

    Of course, this still refers to Xmas - you will need to change this to refer to your birthday.
